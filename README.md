# 🌐 Social Network Platform

A full-fledged social networking platform built with Node.js, WebSockets, HTML, CSS, and JavaScript. Features user profiles, posts, comments, likes, friend requests, and real-time updates.

## ✨ Features

### Core Features
- **User Authentication** - Register and login system
- **User Profiles** - Custom profiles with bio, profile picture, cover photo
- **Create Posts** - Share text and images with your network
- **Like Posts** - Show appreciation for content
- **Comment System** - Engage in discussions on posts
- **Friend System** - Send, accept, and reject friend requests
- **Real-time Updates** - Live notifications and post updates via WebSockets

### Social Features
- **News Feed** - See posts from friends and public profiles
- **Friend Suggestions** - Discover new people to connect with
- **Search Users** - Find friends by name or username
- **Privacy Settings** - Control who sees your content (Public/Friends/Private)
- **Notifications** - Real-time alerts for likes, comments, and friend requests

### Technical Features
- **WebSocket Integration** - Real-time bidirectional communication
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Modern UI** - Gradient backgrounds, glassmorphism, smooth animations
- **RESTful API** - Clean API architecture

## 🛠️ Technologies Used

### Backend
- **Node.js** - JavaScript runtime
- **WebSocket (ws)** - Real-time communication
- **File System (fs)** - Data persistence

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Modern styling with Flexbox/Grid
- **JavaScript (ES6+)** - Dynamic functionality
- **Font Awesome** - Icons
- **Google Fonts** - Poppins typography
